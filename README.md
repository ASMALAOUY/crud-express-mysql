##  Description

Application web complète de gestion de produits développée avec **Node.js**, **Express.js**, **MySQL** et **EJS**. Cette application implémente toutes les opérations CRUD (Create, Read, Update, Delete) avec une interface utilisateur moderne et responsive.

###  Fonctionnalités

-  **CRUD complet** - Créer, Lire, Modifier, Supprimer des produits
-  **Recherche avancée** - Filtrer les produits par nom ou description
-  **Pagination** - Navigation efficace à travers les produits
-  **Design moderne** - Interface utilisateur responsive avec Bootstrap 5
-  **Mobile-friendly** - Adaptation automatique à tous les appareils
-  **Sécurisé** - Protection contre les injections SQL et validation des données
-  **Messages flash** - Notifications de succès/erreur
-  **Statistiques** - Affichage du nombre total de produits
